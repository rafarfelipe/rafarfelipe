## 💼 **Experiência & Competências-Chave**

### **Test Automation Architecture**
- ✅ Desenho de **frameworks do zero** (Cypress, Playwright, Selenium)
- ✅ **Page Object Model** + **Screenplay Pattern** + **Data-Driven Testing**
- ✅ Execução paralela, retry logic, flaky test detection

### **CI/CD & Quality Gates**
- ✅ Pipelines **GitHub Actions / GitLab CI** multi-stage (lint → unit → api → e2e → performance)
- ✅ **Quality Gates**: coverage thresholds, mutation testing, contract testing, a11y audit
- ✅ Deploy preview com testes automatizados (PR environments)

### **Testing Strategies**
- ✅ **Shift-Left**: Testes unitários → Contrato → Integração → E2E → Performance
- ✅ **Contract Testing** (Pact) entre microserviços
- ✅ **Visual Regression Testing** (Percy, Playwright Snapshot)
- ✅ **Accessibility Testing** automatizado (axe-core, WCAG 2.1 AA)

### **Leadership & Mentoring**
- ✅ Code review com foco em **testabilidade**
- ✅ Workshops de **TDD/BDD** para devs
- ✅ Definição de **Definition of Done** com qualidade embutida
- ✅ Métricas de qualidade (DORA, Change Failure Rate, MTTR)
